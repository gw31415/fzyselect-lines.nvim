# fzyselect-lines.nvim

Jumping-lines source of [fzyselect.vim](https://github.com/gw31415/fzyselect.vim).

## Requirements

- [fzyselect.vim](https://github.com/gw31415/fzyselect.vim)
- Neovim

## Usage

Call `require 'fzyselect-lines'.open()` and jump.
